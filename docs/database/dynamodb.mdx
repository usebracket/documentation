## Basics

Our DynamoDB integration is currently live. Users can connect whole collections with tools like Airtable and Notion.

## Allowed fields

The following fields are currently accepted for Bracket's bidirectional syncing:

- String
- Number
- Timestamp (Stored as String in ISO format)
- List
- Map
- Null
