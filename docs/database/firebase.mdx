## Basics

Our Firestore and Realtime Firebase integration is currently live. Users can connect whole collections with tools like Airtable and Notion.

## Allowed fields

The following fields are currently accepted for Bracket's bidirectional syncing:

- string
- boolean
- number
- map
- array
- null
- timestamp
