## Connecting via oauth

1. On the [new connections page](https://app.usebracket.com/), select `airtable`

2. Click "Link new bases"

3. Click "+ Add a base", then "Grant access"

4. Select the base and table you would like to sync with Bracket.

<Info>Bracket also supports syncs using Airtable API keys and personal access tokens, but support for these is deprecated on the web-app. Ping us if you prefer to connect using these methods.</Info>
