## Basics

Google Sheets (GSheets) is one of the most popular and flexible multi-use tools.

With Bracket, you can sync any GSheets database with

1. Any SQL database
2. Any NoSQL database, and/or
3. Airtable/Notion

## Supported and partially supported fields

The following field types are currently supported:

- Text
- Number
- Timestamp
